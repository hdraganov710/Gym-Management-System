# Gym Management System

A Java-based application designed to streamline gym operations, member tracking, and role-based access control.

## Features
* **Role-Based Access**: Separate interfaces for Admins, Coaches, and Clients.
* **Member Management**: Track gym memberships and personal details.
* **In-Memory Persistence**: Efficient data handling for sessions.
* **CLI Interface**: A secure and intuitive command-line interface for system interactions.

## Getting Started
1. Clone this repository: `git clone https://github.com/hdraganov710/Gym-Management-System.git`
2. Open the project in your favorite IDE (IntelliJ recommended).
3. Compile and run the `Main.java` class.
